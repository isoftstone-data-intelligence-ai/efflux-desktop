Efflux 是一个基于大语言模型的 Agent 智能体对话客户端，提供流式会话响应和完整的对话历史管理。通过集成 MCP 协议，系统可作为 MCP Host 连接不同的 MCP Servers，为模型提供标准化的工具调用和数据访问能力。





### 主要特性
- 快速构建 Agent 智能体
- MCP 工具动态加载与调用
- 支持多种大语言模型接入
- 实时流式对话响应
- 会话历史记录管理


### 环境要求
- Python 3.12+
- PostgreSQL
- uv 包和项目管理工具，可通过`pip install uv`安装
